# cXc-Music-Charts
A record of the Daily, Weekly, Monthly and Yearly Top Music charts on cXc Music (music.cxc.world)

# About This Repo
This repo holds 
1. cXc Music SE Global All-time charts
2. cXc Music Beta Gregorian charts (Future)
